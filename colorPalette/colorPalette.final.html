How to Use the Interactive Color Palette Selector?

Welcome to the Interactive Color Palette Selector! This tool allows you to experiment with color combinations to create a beautiful palette for your design projects. Here's how to get started:

  • 1. Select Colors: Choose a color for each element:
    • Wall Color for the background
    • Furniture Color for main features
    • Accent Color for highlights
    • Decor Color for extra touches
  • 2. Apply Colors: Once you've chosen your colors, click “Apply Colors” to see your palette come to life in the preview box.
  • 3. Copy Hex Codes: Hover over each color section to see its hex code. Click "Copy" to save the color code to your clipboard. A checkmark will briefly appear to confirm that it's been copied.

Experiment, mix, and match to find the palette that best fits your style!
